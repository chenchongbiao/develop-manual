◆ scrollTo

void AbstractWheelWidget::scrollTo ( int  index)
slot

Rotates the wheel widget to a given index. You can also give an index greater than itemCount or less than zero in which case the wheel widget will scroll in the given direction and end up with (index % itemCount)
