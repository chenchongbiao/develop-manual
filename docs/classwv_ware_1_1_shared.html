详细描述

Reference counting for shared objects. If you derive your object from this class, then you may use it in conjunction with SharedPtr to control the lifetime of your object.

Specifically, all classes that derive from Shared have an internal counter keeping track of how many other objects have a reference to their object. If used with SharedPtr, then your object will not be deleted until all references to the object have been released.

You should probably not ever use any of the methods in this class directly – let the SharedPtr take care of that. Just derive your class from Shared and forget about it.
