详细描述

A tiny helper class to wrap the PAP and any additional information we want to pass to the consumer. Right now we have a ListInfo object if the paragraph belongs to some list.

◆ listInfo()

const ListInfo * ParagraphProperties::listInfo ( ) const

If this paragraph belongs to a list, the ListInfo object will be valid and contain useful information about the formatting of the list counter. If the paragraph is not inside a list, this method will return 0.
