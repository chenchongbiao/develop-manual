详细描述

The TableHandler class is the interface for all table related callbacks. It informs about about starts and ends of table cells and rows. Only invoking a table row functor triggers the parsing of the given table row, and results in table(Row|Cell)(Start|End) callbacks.

Word doesn't store information about table boundaries internally, but it's quite easy to find out the start/end of a table, as all table row functors are emitted in a row (pun intended). This means that if you get a sequence of table row functors and then suddenly a paragraphStart callback, you know that the table has ended.
