◆ xszFfn

UString FFN::xszFfn

zero terminated string that records name of font. Possibly followed by a second xsz which records the name of an alternate font to use if the first named font does not exist on this system. Maximal size of xszFfn is 65 characters. We are using two UStrings here, the alternative string (xszFfnAlt) will contain the alternative font name in case ixchSzAlt is != 0
