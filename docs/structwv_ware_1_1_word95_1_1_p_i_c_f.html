◆ mfp

METAFILEPICT wvWare::Word95::PICF::mfp

If a Windows metafiles is stored immediatelly followingthe PIC structure, the mfp is a Window's METAFILEPICT structure. When the data immediately following the PIC is aTIFF filename, mfp.mm == 98. If a bitmap is stored after the pic,mfp.mm == 99 When the PIC describes a bitmap, mfp.xExt is the width of the bitmap in pixels and mfp.yExt is the height of the bitmap in pixels..
