详细描述

This class allows to replace the character values of some characters in a Word document, like non-breaking hyphens,... during the processing. As we have to look at every single character anyway, we can as well do single-character replacement in case it's necessary. That way you don't have to scan all the text again in the consumer filter.

We use U8 instead of U16 as all interesting Word characters are within 7bit ASCII range, the names of the methods should be descriptive enough.
