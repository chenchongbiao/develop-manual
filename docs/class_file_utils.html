static

Display human readable file size.

by default, will display size unit. like 1.2 GB or 616 MB. By using forceUnit argument we can also force to display a unit size with the required unit. 0 for bytes, 1 for KiB, 2 for MiB, 3 for GiB, etc.

参数
numThe file size number.
withUnitVisibleSize unit visible or not.
precisionPrecision for float number after the dot.
forceUnitForce to use a unit.
返回
